Docker相关

CentOS7安装Docker

  • yum list docker //查询可下载的docker
  • yum -y install xxx //下载
  • systemctl start/stop/restart docker

基本命令

相关链接

  • docker ps [OPTIONS]查看容器
  • docker logs [OPTIONS] CONTAINER查看容器日志

安装docker-compose

相关链接

  • curl -L https://github.com/docker/compose/releases/download/1.14.0/docker-compose-\`uname -s`-`uname -m` > /usr/local/bin/docker-compose //下载并安装docker-compose
  • chmod +x /usr/local/bin/docker-compose
  • docker-compose -version

docker内安装命令

  • apt-get update
  • apt-get install vim -y
  • apt-get install yum -y

docker安装postgresql

  • docker pull postgres:10.0
  • docker volume create pgdata
  • docker run -it –name postgres -e POSTGRES_PASSWORD=123456 -p 55432:5432 -v pgdata:/var/lib/postgresql/data postgres:10.0
