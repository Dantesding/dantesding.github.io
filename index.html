java线程实验

###测试CPU指令重排序
##代码
public class ThreadTest_reorder_01 {
private static int x = 0, y = 0, a = 0, b = 0;
public static void main(String[] args) throws Exception{
int i = 0;
while (true){
i++;
x = 0;
y = 0;
a = 0;
b = 0;
Thread one = new Thread(new Runnable() {
@Override
public void run() {
a = 1;
x = b;
}
});
Thread two = new Thread(new Runnable() {
@Override
public void run() {
b = 1;
y = a;
}
});
one.start();
two.start();
one.join();
two.join();
if(x == 0 && y == 0){
System.out.println(“第” + i + “次” + “x=” + x + “y=” + y);
break;
}
}
}
}

结果 第3348518次x=0y=0

CentOS7虚拟机在NAT模式下的固定IP配置

环境说明

虚拟机软件 VMware Workstation Pro 15
ISO文件 CentOS-7-x86_64-DVD-1908.iso 下载地址 阿里云官方镜像站下载地址 选择OS镜像

网络配置

配置IP

cd /etc/sysconfig/network-scripts 进入网络配置目录
vi ifcfg-ensxx 注意选择网卡，我的是ens33
BOOTPROTP=static 改成静态地址
DEFROUTE=yes
IPADDR=xx.xx.xx.xx 可以在VMware编辑＞虚拟网络编辑器查看网段
NETMASK=255.255.255.0
GATEWAY=xx.xx.xx.2
ONBOOT=yes 重启时有效
保存退出

配置域名服务器

vi /etc/resolv.conf
nameserver 119.29.29.29
保存退出

重启网络

systemctl restart network 重启网络
